# Cube-Velocity
نسخة WebGame (pseudo-3D) جاهزة للعمل على الجوال والكمبيوتر.

تشغيل محلي:
- افتح `splash.html` أو `index.html` في مستعرض ويب.

هيكل المشروع:
- files: splash.html, index.html, game.html, gameover.html, shop.html
- styles/style.css
- scripts/: كل السكربتات المذكورة

ملاحظات:
- الإعلانات حالياً محاكاة داخل `scripts/ads.js` — قم بوضع SDK الشبكة التي تفضلها (AdMob / IronSource / وغيرها) لاحقًا.
- لتقليل التأخير: خزّن الأصوات محلياً داخل /assets/audio.
